# image-shrink
ImageShrink is a cross platform desktop application which can reduce size of images built using electron.js 

![alt tag](https://i.imgur.com/0lHP4tV.jpg)

# build command

for windows
```npm run package-win```

for mac
```npm run package-mac```

for linux
```npm run package-linux```

note: run 
```npm install```
before building
